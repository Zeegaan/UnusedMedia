Enhanced media is an umbraco package, that enhances the media section with 2 new dashboards!

- The enhanced media dashboard: You can see all unused media, here you can select the unused media, and delete them.
- The enhanced recycle bin: Here you can see a list of media in the recycle bin, and choose to restore them if you want to.
